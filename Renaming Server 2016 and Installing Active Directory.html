Installing VirtualBox and Server 2016

Description

This project walks through the step-by-step setup of a Windows Server 2016 virtual machine. The lab provides exposure to core help desk and IT support concepts, such as system specs assessment, virtualization, ISO image handling, and preparing for Active Directory configuration.

Lab Environment

  • Platform: Oracle VirtualBox (compatible with VMware and other hypervisors)
  • Operating System: Windows Server 2016 (ISO from Microsoft Evaluation Center)
  • Host Requirements:
    • Minimum 2 GB RAM (recommended 4–8 GB)
    • At least 50 GB available disk space
    • Virtualization enabled in BIOS/UEFI

Installation Preparation

  1. Download and install VirtualBox and the Extension Pack.
  2. Verify hardware specs via system properties. Go to Settings > System > About.
  3. Enable virtualization in BIOS if required.

ISO Acquisition

  1. Go to Microsoft Evaluation Center and download Windows Server 2016.
  2. Click on Download the ISO.
  3. Register and download. Evaluation lasts 180 days.

Installing VirtualBox & Extension Pack

  1. Launch Setup Wizard → Click Next
  2. Accept the License Agreement → Click Next
  3. Choose Installation Components → Click Next
  4. Network Interfaces Warning → Click Yes
  5. Python Bindings Warning → Click Yes or install dependencies later
  6. Select Additional Tasks → Click Next
  7. Ready to Install → Click Install
  8. Installation Progress → Wait for completion
  9. Install Extension Pack → Click Install
  10. Accept License Agreement for Extension Pack → Click I Agree

Virtual Machine Setup

  1. Name and OS:
    • Name: Server-2016-Lab
    • Select ISO file
    • Check Skip Unattended Install → Click Next
  2. Configure Hardware:
    • Base Memory: 8192 MB
    • Processors: 1
  3. Virtual Hard Disk:
    • Disk Size: 50 GB

Windows Server Installation

  1. Choose Language → Click Next
  2. Click Install now
  3. Select version: Standard Evaluation (Desktop Experience) → Click Next
  4. Accept License Terms → Click Next
  5. Select Custom: Install Windows only (advanced)
  6. Choose virtual hard disk → Click Next
  7. Wait for installation to finish
  8. Set Administrator password → Click Finish
  9. Login screen → Use VirtualBox menu: Input > Keyboard > Insert Ctrl+Alt+Del
  10. Server Manager opens → Begin configuration
